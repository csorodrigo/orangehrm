CIA Férias

Este ambiente é iniciado a partir do código local do projeto e não depende de downloads externos para subir a aplicação.

Execução local

  • Execute docker compose -f docker-compose.local.yml up -d --build.
  • Acesse http://localhost:8081/web/index.php/auth/login.
  • Entre com Admin / CiaFerias@2026!.
